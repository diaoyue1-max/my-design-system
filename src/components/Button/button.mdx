## Handoff Rules

Before you ship the design...

1. Always use the instance from the asset panel, never detach the component.
2. Never use more than one Primary button per screen.
3. Ghost buttons should only appear on light backgrounds.
4. For edge cases, contact the design system team before going custom.

## When to Use

- Use **Primary** for the single most important action on a page.
- Use **Secondary** for supporting actions alongside a primary.
- Use **Ghost** for low-emphasis actions or in toolbars.
- Use **Danger** for destructive actions like delete or remove.

## Do Not Use When

- You need a link that navigates — use a text link instead.
- You have more than 3 actions — consider a dropdown menu.
